# SchoolFinder Landing Page

SchoolFinder is a landing page designed to help parents, students and teachers explore and discover schools from Early Childhood Development (ECD) to University level.

The platform provides a simple interface to search, compare and explore schools based on location and level of education.

## Technologies Used

- HTML5
- Tailwind CSS
- CSS Grid
- Flexbox

## Layout Components

### Navigation Bar
A responsive navigation bar that allows users to navigate between sections.

### Hero Section
Introduces the platform

## How It Works
Explains about the platform in three steps:
- Search schools
- Compare schools
- Contact schools

### School Categories
Displays major education levels:
- ECD
- Primary
- Junior schools
- Senior schools
- TVET
- University

### Contact Form
A support form allowing users to send inquiries.
